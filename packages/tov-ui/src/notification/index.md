## 属性
| 属性     | 类型            | 描述                             | 默认值   |
|----------|-----------------|----------------------------------|----------|
| title    | `string`        | 通知的标题                       | `无`     |
| content  | `string | VNode` | 通知的正文内容                   | `无`     |
| duration | `number`        | 通知显示的持续时间（毫秒）      | `3000`   |
| appContext | `any`        | 应用上下文（可选）               | `无`     |
