## 属性
									
| 参数 | 说明 | 类型 | 默认值  |
| -------- | ----------------- | --------- | --- |
| placeholder | 控制显示的位置 | `top` `left` `right` `bottom` `topLeft` `topRight` `bottomLeft` `bottomRight` `leftTop` `leftBottom` `rightTop` `rightBottom`| 	`bottom`  |
| trigger | 触发方式 | `hover` `click`  | `hover` |
| content | 显示的内容，也可以通过 `slot` 传入 DOM | `string`  | -   |
